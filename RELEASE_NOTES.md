# Release Notes - WorldExplorer

## Version 1.1.0

### 🌍 English

**New Features:**
- ✨ **Offline-First Country Data**: All country information is now stored locally as JSON. Explore the world without internet connection!
- 🚀 **Improved Performance**: Eliminated API calls for instant country lookups and searches
- 🔒 **Better Reliability**: No dependency on external API availability

**Fixes & Improvements:**
- Fixed daily country card rendering with proper localization
- Resolved map tooltip flag display issues
- Enhanced defensive data validation to prevent crashes
- Fixed Android app bundle signing for Google Play Store compatibility

**Technical:**
- Migrated from REST Countries API to embedded local dataset (250 countries)
- All country names now properly localized in English, German, Spanish, and Polish
- Improved error handling and data validation
